> Source: Deep research from ExtensionPay, Starter Story, Indie Hackers, real revenue data

## Payment System: ExtensionPay

- **Why**: Purpose-built for Chrome extensions. No backend server needed.
- **Fees**: 5% + Stripe's 2.9% + $0.30 = ~92% take-home
- **Supports**: One-time, monthly, yearly, free trials
- **Proven**: $500K+ processed for extension creators
- **Setup**: Add JS library, configure dashboard, done

**Alternative**: LemonSqueezy for license-key model (5% + $0.50, handles global tax)
**Later**: Stripe direct only if building web dashboards alongside extensions

## Pricing Strategy

### Subscription Tiers
| Tier | Monthly | Annual (20% off) | Target |
|------|---------|-------------------|--------|
| Utility | $2.99-3.99 | $29-39/yr | Simple tools (ClipUnlock, CookieForge, DarkShift) |
| Productivity | $4.99-5.99 | $49-59/yr | Feature-rich (FocusForge, TabVault, PageDigest, SnipVault) |
| Professional | $6.99-9.99 | $69-99/yr | Business tools (QuickCapture, PriceRadar) |

### Why Subscription > One-Time
- 30% higher retention
- Exit multiples: 40-60x monthly profit (vs much lower for one-time)
- Rick Blyth achieved 57x multiplier on exit
- Predictable MRR for planning

### Why NOT Lifetime Deals
- Permanent support obligation
- Zero recurring revenue
- Damages exit value
- Only exception: limited 48-hour launch promo to generate reviews, then kill it

## Free vs Paid Split (Golden Rule)

**Free tier must be genuinely useful — NOT crippled.** Users uninstall crippled free tiers before getting hooked.

### What to Lock Behind Paid
- Export (CSV, PDF, JSON)
- Cloud sync across devices
- Advanced filters and search
- Third-party integrations
- Customization and themes
- Bulk operations
- History/analytics beyond basic window
- Team/collaboration features

### What Stays Free
- Core problem-solving functionality
- Basic usage that demonstrates value
- Enough to form a habit and leave a review

## Conversion Benchmarks

| Metric | Benchmark |
|--------|-----------|
| Free → Paid conversion | 2-5% (top performers 5-10%) |
| Freemium install boost | 5-7x more installs than paid-only |
| Optimal monthly price | $5-9/mo for most solo dev extensions |
| Annual discount | 15-20% over monthly |
| Target for $1K MRR | 100 users at $10/mo OR 200 users at $5/mo |

## Revenue Math

### Scenario A: 10,000 Free Users (achievable in 6-12 months)
- 3% conversion = 300 paying users
- $5/mo average = **$1,500 MRR**
- 80% margin = **$1,200/mo profit**
- Exit value (50x) = **$60,000**

### Scenario B: 50,000 Free Users (achievable in 12-18 months)
- 3% conversion = 1,500 paying users
- $5/mo average = **$7,500 MRR**
- 80% margin = **$6,000/mo profit**
- Exit value (50x) = **$300,000**

### Portfolio of 10 Extensions (Scenario A each)
- 100,000 total free users
- 3,000 paying users
- **$15,000 MRR combined**
- **$12,000/mo profit**
- **Exit value: $600,000**

## Key Rules

1. Never launch paid-only — free tier is your growth engine
2. Never do lifetime deals (except 48-hour launch promo)
3. Always offer annual pricing alongside monthly
4. Target professionals who use your tool to make/save money
5. Build for a niche first, expand later
6. Cross-promote all extensions from each extension's settings page
7. Translate listings to 5+ languages = 5x search surfaces
8. Respond to EVERY review (positive and negative) — shows active development
