WHY COPYUNLOCK?

Unlike simple copy unlockers that only remove CSS user-select, CopyUnlock detects restrictions across 5 categories:

1. CSS blocks — user-select:none, pointer-events:none, -webkit-touch-callout
2. JavaScript handlers — oncopy, onpaste, oncut, oncontextmenu, onselectstart, onkeydown
3. DOM watchers — MutationObserver re-blockers, copy event overrides
4. Clipboard API interception — document.execCommand, navigator.clipboard.writeText
5. Visual overlays — transparent divs, fixed-position elements blocking text selection

Most copy protection uses 3-5 of these simultaneously. Other extensions only handle 1-2 categories and fail on heavily protected pages. CopyUnlock handles all 29 methods across all 5 categories.
