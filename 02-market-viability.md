## Discovery: How Users Find Extensions

1. **Chrome Web Store Search** (primary) -- SEO-based, like Etsy
   - Install velocity (strongest signal)
   - Active users / total installs (retention)
   - Click-through rate from search
   - View-to-install conversion
   - Uninstall rate (penalty)
   - Rating quality + quantity
   - Regular updates
2. **Category browsing** -- sorted by popularity
3. **External referrals** -- blog, YouTube, Product Hunt, Reddit
4. **Featured/Editorial** -- rare, not the main channel

**Key hack**: Translate listing into multiple languages = multiple search surfaces

## Volume vs. Focus Strategy

**Volume (3-5 extensions)**: More search surface area. Test market demand. Kill losers, double down on winners.
- S-Hub developer: 17 extensions x 8 languages = 136 searchable listings
- Adil: Built 20, sold 4 in a month

**Focus (1-2 extensions)**: Deeper quality. Better retention. Higher review velocity.
- Rick Blyth: 2 main extensions -> $500K+
- GoFullPage: 1 feature -> $120K/yr

**Optimal**: Build 3-5 quickly, ruthlessly focus on the 1-2 that show traction.

## Economics

- Developer fee: $5 one-time (unlimited extensions)

- Profit margins: 70-85%
